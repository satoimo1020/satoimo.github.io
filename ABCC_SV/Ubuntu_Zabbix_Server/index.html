Zabbix サーバー構築手順

test1（Zabbixサーバー）に、Zabbixサーバー、Webフロントエンド、データベース（MySQL）を構築する

1-1
Zabbixリポジトリの登録

Zabbix公式のリポジトリをシステムに追加する。wgetで取得するファイルはインストール後、不要になるため tmp ディレクトリにダウンロードする

Terminal
cd /tmp
wget https://repo.zabbix.com/zabbix/7.0/ubuntu/pool/main/z/zabbix-release/zabbix-release_latest_7.0+ubuntu24.04_all.deb
TIP
Ubuntu 24.04用のZabbix 7.0公式リポジトリパッケージをダウンロード
sudo dpkg -i zabbix-release_latest_7.0+ubuntu24.04_all.deb
TIP
ダウンロードしたリポジトリパッケージをシステムにインストール
sudo apt update
TIP
新しく追加されたZabbixリポジトリのパッケージリストを同期
1-2
必要なパッケージをインストール

Zabbixサーバー、フロントエンド、Apache、およびZabbixエージェントを一括でインストール

⚠️ mysql-serverに関する注意事項
Ubuntu 24.04にZabbix用のMySQLを導入する際、すでに新しいバージョンのMySQLが存在する状態で、それよりも古いバージョンのMySQLを上書きインストールしようとすると、データの互換性がなくなりシステムがフリーズモードに移行するため、新たに mysql-server はインストールしないように（今回は既にインストール済みのため除外）
Terminal
sudo apt install -y zabbix-server-mysql zabbix-frontend-php zabbix-apache-conf zabbix-sql-scripts zabbix-agent2
TIP
Zabbixに必要な各種コンポーネントをインストール（-yで自動承認）
1-3
データベースの初期設定

MySQLにログインし、Zabbix用のデータベースとユーザーを作成する

Terminal
sudo mysql
TIP
MySQLにroot権限でログイン（パスワードレス、または設定したrootパスワードを入力）
MySQL プロンプト内でのSQL実行

※ 'password' 部分は任意のパスワードに置き換えて設定

create database zabbix character set utf8mb4 collate utf8mb4_bin; create user zabbix@localhost identified by 'password'; grant all privileges on zabbix.* to zabbix@localhost; set global log_bin_trust_function_creators = 1; quit;
1-4
初期データのインポート

Zabbixが提供する初期スキーマとデータを、先ほど作成したデータベースに流し込む

Terminal
zcat /usr/share/zabbix-sql-scripts/mysql/server.sql.gz | mysql --default-character-set=utf8mb4 -uzabbix -p zabbix
TIP
パスワードを求められるため、[1-3]で設定した zabbix ユーザーのパスワードを入力。数分程かかる場合がある

インポートが終わったら、データベースの関数作成に関する一時設定を元に戻す

Terminal
sudo mysql
MySQL プロンプト内でのSQL実行
set global log_bin_trust_function_creators = 0; quit;
1-5
Zabbixサーバーの設定編集

Zabbixサーバーの構成ファイルを編集し、データベースのパスワードを指定

① 構成ファイルの編集
sudo nano /etc/zabbix/zabbix_server.conf

ファイル内で `DBPassword=` の行を探して、コメントアウト（#）を外し、データベースのパスワードを指定

DBPassword=password
TIP
[1-3]で設定した zabbix ユーザーのパスワードを記述
1-6
サービスの起動と有効化

各種サービスを起動し、OS起動時に自動実行されるよう設定

Terminal
sudo systemctl restart zabbix-server zabbix-agent2 apache2 mysql
sudo systemctl enable zabbix-server zabbix-agent2 apache2 mysql
1-7
WebUIでの初期セットアップ

ホストOSのブラウザから、以下のURLにアクセスする。

http://192.168.56.201:8080/zabbix

💡 セットアップウィザードの手順
  1. Welcome: 言語を選択（多分英語しか選べない）して「Next step」
  2. Check of pre-requisites: 全て「OK」であることを確認して「Next step」
  3. Configure DB connection: Password 欄に [1-3] で決めたパスワードを入力して「Next step」
  4. Settings: 任意のサーバー名を入力して「Next step」
  5. Install: 設定内容を確認して「Finish」
🔐 初期ログイン情報
  • ユーザー名: Admin (※イニシャルが大文字になっているので注意)
  • パスワード: zabbix
