CDの音楽をスマートフォンにいれる

1. PCに光学ドライブを接続する

PCを起動し、光学ドライブを接続しておきます。

2. Windows Media Playerを開く

検索窓に ”Windows Media Player” と入力して開きます。

3. CDを光学ドライブに入れて読み込む

Windows Madia Playerが開いたら、光学ドライブにCDを入れて読み込みます。

読み込み中...

4. 取り込みのオプション設定

CDの読み込みが終わると内容が表示されます。

次に、画像のように上部の “取り込みの設定” → “形式” → “MP3” に設定して、チェックが入っていることを確認します。

5. CDの取り込み

CDの取り込みをクリックして、すべて取り込み終わるまで待ちます。

6. スマートフォンに入れる

エクスプローラーを新しく開き、Musicフォルダを開きます。

スマートフォンとPCを接続します。

もう1つエクスプローラーが立ち上がるので、内部共有ストレージ→Musicの中に取り込んだ音楽をコピーします。

7. スマートフォンにアプリを入れる

今回はYouTube Musicを使いますが、既にインストールされている場合は飛ばして構いません。

スマートフォンで、プレイストアからYouTube Musicをインストールします。
