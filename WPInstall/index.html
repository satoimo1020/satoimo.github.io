WordPress構築手順書

このドキュメントは、Linuxサーバー上にApache、MariaDB、PHPをインストールし、WordPressをセットアップするための手順をまとめたものです。

1. システム更新とパッケージのインストール

sudo apt update
sudo apt install apache2 mariadb-server php libapache2-mod-php php-mysql php-curl php-gd php-mbstring php-xml php-xmlrpc php-soap php-intl php-zip -y

2. データベースのセットアップ

sudo mysql -u root

-- MySQLコンソール内での操作
CREATE DATABASE wordpress_db;
CREATE USER 'test'@'localhost' IDENTIFIED BY 'test';
GRANT ALL PRIVILEGES ON wordpress_db.* TO 'test'@'localhost';
FLUSH PRIVILEGES;
EXIT;

3. WordPress本体のダウンロードと配置

cd /tmp
wget https://ja.wordpress.org/latest-ja.tar.gz
tar -xzvf latest-ja.tar.gz
sudo cp -r wordpress/* /var/www/html/
sudo rm /var/www/html/index.html

4. 権限の設定

sudo chown -R www-data:www-data /var/www/html/
sudo chmod -R 755 /var/www/html/
補足: 設定を反映させるため、Apacheを再起動することをお勧めします。
sudo systemctl restart apache2

5. ブラウザでの初期設定

注意点：ポートフォワーディングを行っているため、ブラウザでアクセスする際は :8080 をアドレスの直後につけること！

ブラウザから http://127.0.0.1:8080 にアクセスし、以下の情報を入力してください。

項目 設定値
データベース名 wordpress_db
ユーザー名 test
パスワード test
データベースのホスト localhost
